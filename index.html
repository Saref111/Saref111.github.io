Все работы в разделе верстки валидны, кроссбраузерные, доступные и сверстаны семантично.

  • Mishka. Милые штуки ручной работы

    Адаптивный(мобильный, планшетный, десктопный) кроссбраузерный сайт. Главная страница, страница каталог и страница формы. Есть модалное окно выбора размера. HTML5, CSS3, JavaScript, Flexbox, Gulp, Git, Less, BEM, SVG sprite. Сайт оптимизирован, файлы минимизированы, изображения загружаються согласно характеристикам экрана пользователя. При возможности для изображений используется формат webp. Оптимизирована загрузка шрифтов.

    Компонентный подход, ООП, паттерн MVC, ES6, Webpack, REST API, Promise, fetch. Приложение умеет добавлять новые точки маршрута, редактировать их, фильтровать, сортировать, удалять. Показана статистика по всем точкам маршрута. Функционал шапки не реализован.
